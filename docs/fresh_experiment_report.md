# 共享单车租借量预测实验报告

## 1. 实验目标

本实验根据 `codex.md` 的任务说明，使用 `data/train.csv` 训练回归模型，对 `data/test.csv` 中每条小时级记录预测共享单车租借量 `cnt`，并生成符合提交格式的结果文件：

```csv
ID,cnt
13904,280
13905,278
```

评价指标按说明采用均方误差 MSE。

## 2. 数据概况

本次实验只读取当前 `data` 目录中的真实数据：

| 数据集 | 行数 | 列数 | ID 范围 | 日期范围 |
| --- | ---: | ---: | --- | --- |
| train.csv | 13903 | 15 | 1-13903 | 2011-01-01 至 2012-08-07 |
| test.csv | 3476 | 14 | 13904-17379 | 2012-08-07 至 2012-12-31 |

训练集目标列 `cnt` 的统计特征如下：

| 指标 | 数值 |
| --- | ---: |
| 均值 | 174.64 |
| 标准差 | 166.96 |
| 最小值 | 1 |
| 中位数 | 130 |
| 最大值 | 957 |

训练集和测试集均无缺失值。测试集从训练集最后一条记录之后连续开始，因此验证集采用时间顺序切分，而不是随机切分。

## 3. 特征工程

模型输入共 52 个特征，主要包括：

1. 原始结构化特征：`season`、`yr`、`mnth`、`hr`、`holiday`、`weekday`、`workingday`、`weathersit`、`temp`、`atemp`、`hum`、`windspeed`。
2. 日期特征：日期序号、日、年内第几天、年内第几周、月初、月末。
3. 周期编码：小时、星期、月份、年内日期的正弦和余弦编码。
4. 出行行为特征：凌晨低需求、早高峰、晚高峰、工作日早晚高峰、非工作日白天。
5. 天气交互特征：温度差、温度平方、湿度平方、风速平方、温湿度交互、温度风速交互、恶劣天气、舒适天气。
6. 历史统计特征：按小时、工作日与小时、月份与小时、天气与小时、季节与小时、星期与小时统计的训练集历史均值。

历史统计特征在验证时只使用验证集之前的训练段计算，避免把未来验证目标泄露给模型；最终训练测试集时才使用完整训练集计算统计量。

## 4. 验证方法

由于测试集是训练集之后的未来时间段，实验使用训练集最后 20% 作为验证集：

| 切分 | 行数 |
| --- | ---: |
| 训练段 | 11122 |
| 验证段 | 2781 |

这种划分更接近真实提交场景：用过去预测未来。

## 5. 模型设计

本次从头实现了一个单文件训练管线 `main.py`，不依赖旧 `src` 实现。模型组包括：

| 模型 | 目标 |
| --- | --- |
| LightGBM | 直接预测 `cnt` |
| LightGBM log | 预测 `log1p(cnt)` 后反变换 |
| XGBoost | 直接预测 `cnt` |
| CatBoost | 直接预测 `cnt` |

在验证集上对四个模型预测结果进行步长为 0.05 的权重网格搜索，选择 MSE 最低的加权融合。

## 6. 验证结果

| 模型 | MSE | RMSE | MAE | R2 |
| --- | ---: | ---: | ---: | ---: |
| LightGBM | 4102.118 | 64.048 | 43.239 | 0.9129 |
| LightGBM log | 5042.658 | 71.012 | 48.485 | 0.8929 |
| XGBoost | 3913.693 | 62.560 | 42.420 | 0.9169 |
| CatBoost | 4636.662 | 68.093 | 46.770 | 0.9015 |
| 融合模型 | 3901.896 | 62.465 | 42.249 | 0.9171 |

最优融合权重为：

| 模型 | 权重 |
| --- | ---: |
| LightGBM | 0.15 |
| LightGBM log | 0.00 |
| XGBoost | 0.80 |
| CatBoost | 0.05 |

融合模型比单个 XGBoost 略有提升，最终用于生成提交文件。

## 7. 输出文件

本次实验生成了以下文件：

| 文件 | 说明 |
| --- | --- |
| `main.py` | 从头实现的完整训练、验证、融合和预测脚本 |
| `output/fresh_experiment_metrics.json` | 本次实验的数据概况、特征列表、验证指标和融合权重 |
| `output/submission_fresh_ensemble.csv` | 最终提交文件，列名为 `ID,cnt` |

提交文件共 3476 行预测结果，无负数预测，无重复 ID。

## 8. 运行方式

在项目根目录运行：

```powershell
python .\main.py
```

脚本会自动读取 `data/train.csv` 和 `data/test.csv`，重新训练模型，并覆盖生成：

```text
output/submission_fresh_ensemble.csv
output/fresh_experiment_metrics.json
```
